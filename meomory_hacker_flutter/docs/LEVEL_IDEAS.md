# Handcrafted Level Ideas

## Design Rules
- Introduce one new demand at a time
- Reuse mechanics with tighter timing before adding another mechanic
- Keep names thematic so progression feels authored

## Levels
1. **Boot Sequence**
   3x3 grid. Reveal 3 target tiles with generous timing. Pure tutorial level that teaches tap-to-recall.

2. **Signal Trace**
   3x3 grid. Reveal 4 targets in a simple shape. Slightly shorter reveal to encourage chunking.

3. **Ghost Cache**
   3x3 grid. Reveal 4 targets, then briefly flash one decoy during hide transition. Teaches focus under distraction.

4. **Firewall Handshake**
   4x4 grid. Reveal 5 targets with a stricter input timer. First level where time pressure matters.

5. **Null Sector**
   4x4 grid. Reveal 6 targets arranged asymmetrically. Designed to break easy corner-based memorization.

6. **Packet Drift**
   4x4 grid. Reveal targets in sequence order. Player must tap in the same order after hide.

7. **Mirror Break**
   4x4 grid. Pattern looks visually symmetrical but contains one intentional asymmetry. Tests overconfidence.

8. **Noise Injection**
   5x5 grid. Reveal 6 targets while non-target nodes animate lightly in the background. Visual noise increases difficulty without changing core rules.

9. **Trace Collapse**
   5x5 grid. Reveal 7 targets with a short timer and one allowed mistake. Encourages risk management and fast recovery.

10. **Core Breach**
    5x5 grid. Multi-phase finale: round one is standard recall, round two is ordered sequence on the same board theme. Feels climactic without requiring a new system.

## Reusable Modifier Catalog
- shorter reveal duration
- shorter input duration
- decoy flash
- ordered tap sequence
- limited mistakes
- visual noise layer
- asymmetric pattern bias

## Tuning Advice
- Difficulty should mostly rise through memory load and timing, not input gimmicks
- If a level feels unfair, reduce targets before adding more tutorial text
- Reserve 5x5 density for later levels so visual clarity stays strong early
